## How to Run

### Requirements
- Node.js 18+ installed
- npm installed

### Steps

1. Open your terminal and go to this folder:
```
cd debale
```

2. Install dependencies:
```
npm install
```

3. Start the development server:
```
npm run dev
```

4. Open your browser at:
```
http://localhost:5173
```
